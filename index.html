Projects

Weather Mobile App

The Weather Mobile App is a project that I developed as part of my personal interest in software development. This Android application is dedicated to providing users with accurate weather forecasts based on their selected locations. The app allows users to stay updated with current weather conditions, hourly and daily forecasts, ensuring they are well-prepared for any weather changes.

Key Features:

  • Accurate weather information based on user-selected location.
  • Hourly and daily weather forecasts for better planning.
  • User-friendly interface for easy navigation.
  • Real-time weather updates.

The Weather Mobile App project provided me with valuable experience in mobile app development, API integration, and user interface design. I thoroughly enjoyed working on this project and look forward to exploring more opportunities to enhance the app's functionality in the future.
